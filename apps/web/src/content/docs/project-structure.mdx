## Overview

AI Stack is organized as a TypeScript monorepo using Turborepo, containing the documentation site, CLI tool, backend services, and reusable packages for AI functionality.

## Configuration Files

### `turbo.json`

Turborepo configuration for monorepo orchestration:

```json
{
  "pipeline": {
    "build": {
      "dependsOn": ["^build"],
      "outputs": ["dist/**", ".next/**"]
    },
    "dev": {
      "cache": false
    },
    "lint": {},
    "typecheck": {}
  }
}
```

### `docker-compose.yml`

Development environment services:

```yaml
version: '3.8'

services:
  postgres:
    image: postgres:15
    environment:
      POSTGRES_DB: aistack
      POSTGRES_USER: postgres
      POSTGRES_PASSWORD: postgres
    ports:
      - "5432:5432"

  qdrant:
    image: qdrant/qdrant
    ports:
      - "6333:6333"
```
