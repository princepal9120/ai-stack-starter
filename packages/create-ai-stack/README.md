# create-ai-stack

The quickest way to create AI Stack applications.

## Usage

```bash
npx create-ai-stack@latest my-app
```

Or with other package managers:

```bash
# pnpm
pnpm create ai-stack@latest my-app

# bun
bun create ai-stack@latest my-app
```

## Options

See the [CLI documentation](https://ai-stack.dev/docs/cli) for all available options.
